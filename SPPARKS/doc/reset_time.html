reset_time command

Syntax: 

reset_time time
  • time = new time

Examples: 

reset_time 0.0
reset_time 100.0

Description:

Set the current time to the specified value. This can be useful if a preliminary run was performed and you wish to reset the time before performing a subsequent run.

Restrictions: none

Related commands: none

Default: none
